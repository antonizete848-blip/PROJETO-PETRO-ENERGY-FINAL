# Petro Energy

Projeto desenvolvido em Angular com integração entre front-end e back-end utilizando JSON Server.

## Tecnologias utilizadas

- Angular
- TypeScript
- JSON Server
- HTML
- CSS

## Funcionalidades

- SPA (Single Page Application)
- Rotas dinâmicas
- Comunicação com back-end
- Consumo de API
- Página de detalhes dinâmica
- Navegação sem recarregamento

## Como executar o projeto

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```

Frontend:

```txt
http://localhost:4200
```

API fake:

```txt
http://localhost:3000
```
